jQuery Navobile

a jQuery plugin that makes mobile navigation easy

Usage

Default usage for Navobile is as easy as this:

$('#foobar').navobile();

You will also need to include the css in the src folder.

Params

Navobile has a few params to customise for you own needs.

  • cta - the element that will open the navigation on click (default: '#show-navigation')
  • content - the selector that wraps the content that will slide out (default: '#content')
  • easing - jQuery easing function to use in $.animate fallback (default: 'linear') for more easing options you will require jQuery.easing
  • changeDOM - Boolean, whether the plugin needs to move the navigation in the DOM structure (default: false)

Any changes to the defaults should be passed in at init

$('#foobar').navobile({
    'cta': '#foo-cta',
    'easing': 'easeQuintOut'
});

Dependancies

Navobile has a couple of dependancies to make full use of its features and get the very best performance possible.

These dependancies are:

  • Modernizer
  • Hammer JS (and specialevents plugin for jQuery)

The versions of the above that were used in development are included in the lib folder of the project.
